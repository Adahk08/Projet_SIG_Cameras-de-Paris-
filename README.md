# Projet de Visualisation des Caméras de Surveillance de Paris

## Description
Ce projet consiste à développer une application desktop en Python avec **Tkinter** permettant la gestion et la visualisation de caméras de surveillance de la ville de Paris.

L’objectif principal est de faciliter l’intégration et la gestion de données géographiques liées aux caméras de surveillance à travers une interface simple et intuitive.

---

## Fonctionnalités

- Interface graphique développée avec **Tkinter**
- Connexion à une base de données
- Authentification utilisateur
- Ajout manuel de nouvelles caméras
- Import automatique de données via des fichiers **KML**
- Insertion des données importées dans la base de données via un bouton *Importer*
- Gestion et stockage des informations géographiques

---

## Technologies utilisées

- Python
- Tkinter
- PostgreSQL / PostGIS
- SQL
- KML
- Bibliothèques Python de traitement de données géographiques

---

## Objectif du projet

Ce projet a été réalisé dans le cadre d’un apprentissage autour :
- des systèmes d’information géographique (SIG),
- de la gestion de bases de données spatiales,
- du développement d’interfaces graphiques,
- et de l’automatisation de l’intégration de données géographiques.

---

## Perspectives d’amélioration

- Affichage cartographique interactif
- Filtrage et recherche des caméras
- Visualisation temps réel
- Export des données
- Déploiement web de l’application
